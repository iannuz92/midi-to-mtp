WORKFLOW

1. Add .mid files with + ADD (multiple selection supported).

2. Set the starting index (e.g. 01).

Each file is saved as pattern_NN.mtp with NN auto-incremented.

3. Choose Resolution and Instrument.

4. Press CONVERT — files are downloaded directly.

5. Copy the .mtp files to the SD card:
/projects/<project_name>/patterns/


RESOLUTION

Sets the minimum quantization step for notes:

1/16 → minimum step = sixteenth note (default)

1/8 → minimum step = eighth note

1/4 → minimum step = quarter note

Use 1/16 for most MIDI files.


INSTRUMENT

Instrument slot (0-47) assigned to all notes.

Corresponds to instruments loaded in the project on the Polyend Tracker (slot 0 = first instrument).


CHORDS – CHORD SPREADING

When multiple notes land on the same step, they are spread across consecutive tracks, matching the Polyend Tracker firmware behaviour when receiving chords via MIDI IN.


Example – C major chord (C4+E4+G4) on step 0:

Track 0, step 0 → G4 (highest voice)

Track 1, step 0 → E4

Track 2, step 0 → C4 (lowest voice)


Maximum polyphony: 11 simultaneous voices.


MERGE MODE

Multiple MIDI files are combined into a single .mtp pattern. Each file is placed on a separate tracker track (0-7). Monophonic per track: if multiple notes fall on the same step, the first wins. Maximum 8 files per pattern.


MIDI FILE TYPES

Type 0 (single track):

Each MIDI channel = separate source for chord spreading.

Type 1 (multi track):

Each MIDI track = separate source.


PATTERN LENGTH

Calculated automatically and rounded up to the nearest multiple of 16: 16, 32, 64, 128 (max).
